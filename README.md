# 解悠科技有限公司实习生周报系统

本版本新增：

1. 管理员日报管理：可新增、编辑、删除任意实习生日报。
2. 任务总表管理：可新增、编辑、删除任务，并审核实习生认领申请。
3. 导出功能：管理员可导出日报 CSV；老板和管理员可导出周报 CSV。
4. 日期区间生成周报：管理员可选择单个实习生或全部实习生，按日期区间一键从日报生成/覆盖周报。
5. 权限保持隔离：实习生只看自己的日报和任务；管理员看日报/任务总表；老板只看周报。
