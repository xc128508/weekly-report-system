# 项目目录说明

## 核心目录

- src/
  - 后端代码，主要文件：src/server.js
- public/
  - 前端静态资源，包括 CSS 和页面增强 JS
- data/
  - 数据文件，主要文件：data/db.json
- node_modules/
  - Node.js 依赖，不要手动删除
- _archive/
  - 历史备份和清理归档文件

## 核心文件

- src/server.js
  - 系统页面、接口、权限、业务逻辑
- public/styles.css
  - 页面样式
- data/db.json
  - 用户、任务、日报、周报等数据
- package.json
  - 项目依赖配置

## 注意

不要删除：
- src/
- public/
- data/
- node_modules/
- package.json
- package-lock.json
